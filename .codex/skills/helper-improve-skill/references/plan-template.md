# Plan Template

Save improvement plans to:

- `AgentHelper/ProjectFiles/SkillImprovementPlans/<skill-name>-improvement-YYYY-MM-DD.md`

Use this structure:

```md
# <skill-name> Improvement Plan

- Date: YYYY-MM-DD
- Target skill: `<skill-name>`
- Reviewer skill: `helper-improve-skill`

## Confirmed Intent

- ...

## Current Skill Summary

- ...

## Findings

1. ...
2. ...

## Proposed Changes

1. ...
2. ...

## Validation Plan

- ...

## Recommended Next Step

- ...
```

Guidelines:

- separate confirmed user intent from your inferred recommendations
- keep findings concrete and behavior-focused
- avoid filler sections when there is little to change
