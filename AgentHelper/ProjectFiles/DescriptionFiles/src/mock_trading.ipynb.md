# `src/mock_trading.ipynb`

This notebook is an exploratory analysis companion to `src/mock_trading.py`. It defines portfolio-metric helpers, loads return data through `src/mock_trading_utils.py`, compares cumulative PnL across forecast horizons, and includes a simple AR(1) baseline alongside saved model position files.

The notebook is analysis code rather than reusable library code. Its outputs are plots and summary tables for mock trading experiments, including Sharpe ratio, drawdown, annualized return, annualized volatility, and neutral trading cost.

Category: exploratory notebook.
